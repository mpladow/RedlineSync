# Jira Ticket Reader & Manager

## Instructions for Codex
1. Fetch the ticket details (including description, comments, and checklist) before writing any code. The board used will be the Resonance Sync Trello board and the tickets will be in the "Development" column.
2. Carefully analyze the Acceptance Criteria and the User Story documented on the Jira card.
3. Before proceeding with any code generation, provide a summary of the ticket, including the Acceptance Criteria and any relevant details. Ask for confirmation that the summary is correct before proceeding with code generation.
